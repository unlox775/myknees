# Extension Icons

This directory should contain the following icon files for the Chrome extension:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

## Icon Requirements

- PNG format
- Transparent background recommended
- Should be recognizable at small sizes
- Follow Chrome extension icon guidelines

## Temporary Icons

For development, you can create simple placeholder icons or use any 16x16, 48x48, and 128x128 PNG files.

## Creating Icons

You can create icons using:
- Online icon generators
- Design tools like Figma, Sketch, or Adobe Illustrator
- Icon libraries like Material Design Icons or Font Awesome

## Example

If you have icon files, place them here:
```
src/icons/
├── icon16.png
├── icon48.png
└── icon128.png
```
